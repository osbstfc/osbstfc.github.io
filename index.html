Obsidian Star Bot (OSB) is a free to use bot for Star Trek Fleet Command

OSB is a free non-invasive, bot for the desktop version (Windows PC) of the online game Star Trek Fleet Command (STFC BOT).

Non-invasive - it does not interact directly with STFC servers in any way. It does not inject code, modify the game interface or do anything whatsoever to the game itself.

Non-detectable – the bot simulates the actions of a human player controlling the mouse and/or keyboard to play the game. It does not leave any traces in the game (like bookmarks) or uses any standard patterns that can be identified.

Will this bot work for me?

This free STFC bot only works on Windows, with the desktop version of STFC.

The bot requires:

  • windows display min. resolution 1920x1080
  • windows display scale set to 100%
  • game language set to English
If you use Ripper's Mod, you need to disable it for the bot to work (rename version.dll to something else and restart the game).
The PC and the internet connection must be good enough to run the game smoothly!
Contact support if requirements are met, but you still have issues with the bot...

Is the bot safe to use?

Yes.

The bot has built-in advanced anti-detection measures. It's safe as long as you are not using it 24/7 (long uninterrupted periods of time) to rise automated red flags.
